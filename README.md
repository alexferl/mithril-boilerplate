# mithril-boilerplate

[Mithril.js](https://mithril.js.org/) boilerplate with [Vite](https://vitejs.dev/) and [Rome](https://rome.tools/).

# Using

## Setup
```shell
npm install
```

## Running in dev
```shell
npm run dev
```

## Lint
```shell
npm run lint
```

## Format
```shell
npm run format
```

## Running in prod preview
```shell
npm run preview
```

## Building for prod
```shell
npm run build
```
