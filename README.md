# mithril-boilerplate

[Mithril.js](https://mithril.js.org/) boilerplate with [Vite](https://vitejs.dev/).

# Using

## Setup

```shell
npm install
```

## Running in dev

```shell
npm run dev
```

## Building for prod

```shell
npm run build
```

## Running in prod preview

```shell
npm run preview
```
